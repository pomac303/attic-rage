In order to build under Win32 you need some glib imports.
Download the following files and unzip them INTO this directory.

http://www.gimp.org/~tml/gimp/win32/glib-2.4.5.zip
http://www.gimp.org/~tml/gimp/win32/glib-dev-2.4.5.zip
http://www.gimp.org/~tml/gimp/win32/gettext-runtime-0.13.1.zip
http://www.gimp.org/~tml/gimp/win32/libiconv-1.9.1.tar.gz

http://www.gimp.org/~tml/gimp/win32/downloads.html for more information about these dependancies.
